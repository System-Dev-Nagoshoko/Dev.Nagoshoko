# Event Guide System

イベント向けに開発された来場者案内システムです。  

来場者が企画や会場情報を簡単に確認できるよう設計されています。

## 🌐 概要

本システムは、イベントにおける来場者向け案内を目的としたWebシステムです。

主な用途：

- 会場マップの表示

- 企画一覧の確認

- 当日の情報案内

- LINE Botとの連携（予定）

## 🚀 機能

- シンプルで見やすい案内UI

- モバイル対応（スマホ最適化）

- QRコードからのアクセス想定

- 軽量構成（高速表示）

## 🛠 技術構成

- HTML / CSS / TypeScript

- GitHub Pages / Cloudflare Pages（予定）

- LINE Messaging API（連携予定）

## ⚠️ 利用について

本ソフトウェアは、特定のイベント案内用途に限定して使用を想定しています。

無断での再配布や、他用途での使用は禁止されています。  

使用を希望する場合は、事前に所有者へご連絡ください。
